Whatsapp Clone provides an in depth view of implementation on how to create a full-stack, mobile, hybrid web application from scratch. This repository covers implementation of 3 major platform's mainly Android, IOS & Web using React Native & MERN stack üî•.

<img src='https://github.com/yuvraj24/whatsapp-clone/blob/master/assets/whclone4.png' height=600 width=1200 /> 

You can see the following features in action in this repository,
* *Instant & Realtime textual One to One chat*
* *User's Online/Offline availability*
* *Last Seen time*
* *Unread Message count indicator*
* *Realtime Status update*
* *Multiple Status view along with progress*
* *Web Login via QR code scan*
